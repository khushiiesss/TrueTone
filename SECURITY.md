# Security Policy

## Supported Versions

Currently, the following versions are being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security related issues, please report them to the maintainers privately rather than opening an issue on GitHub. 

You can contact the project maintainers via email or through private messages on the corresponding platforms. We will strive to acknowledge your report within 48 hours and provide a timeline for fixing the issue.

Please provide the following details in your report:
* A detailed description of the vulnerability.
* Steps to reproduce the vulnerability.
* The potential impact of the vulnerability.
* Any potential mitigation strategies or fixes you recommend.

Thank you for helping keep TrueTone AI secure!
